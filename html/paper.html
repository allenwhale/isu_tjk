Guideline of Abstract

Start the text of the abstract here. The length should not exceed 300 words. Underline the name of attendee, and indicate the corresponding author with a superscript asterisk (*). E-mail address should be included. Use font type Times New Roman, 12 points, with single spaced lines. Page margins should be 2 cm for top and bottom, 2.5 cm for right and left margins with A4-size paper. Only half of one page for an abstract provided. No keywords required. Tables, figures and references should not be included in the abstract. Please save your abstract in a MS Word document (File Name and Message Subject should be: presenting author’s name) and should be submitted online by September 15, 2015. Please indicate at the end of abstract about which section your presentation is related. Only “Section H. English Oral Competition” is oral presentation, others are all poster presentation. Please note that a PDF file of “Copyright transfer for publication ” should also be up-loaded simultaneously.
